## Database set up
``CREATE TABLE `patterns` (
  `id` int(11) unsigned NOT NULL AUTO_INCREMENT,
  `name` varchar(39) NOT NULL,
  `group` varchar(39) NOT NULL,
  `implementation` varchar(39) NOT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB AUTO_INCREMENT=9 DEFAULT CHARSET=utf8``
